# DOM børn

Se på følgende side:

```html
<html>
<body>
  <div>Brugere:</div>
  <ul>
    <li>Lisbeth</li>
    <li>Mikkel</li>
  </ul>
</body>
</html>
```

For hver af de følgende, giv mindst en måde at tilgå dem:
- `<div>` DOM noden?
- `<ul>` DOM noden?
- Den anden `<li>` (med Mikkel)?

# Hent attributten

Skriv en kode der vælger elementet med attributten `data-widget-name` fra dokumentet og læser dens værdi.

```html run
<!DOCTYPE html>
<html>
<body>

  <div data-widget-name="menu">Vælg en genre</div>

  <script>
    /* din kode */
  </script>
</body>
</html>
```

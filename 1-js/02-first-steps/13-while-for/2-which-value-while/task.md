# Hvilke værdier viser while-løkken?

For hver iteration af løkken, skriv ned hvilken værdi den viser, og sammenlign derefter med løsningen.

Viser begge løkker de samme værdier, eller ikke?

1. Prefix-formen `++i`:

    ```js
    let i = 0;
    while (++i < 5) alert( i );
    ```
2. Postfix-formen `i++`

    ```js
    let i = 0;
    while (i++ < 5) alert( i );
    ```

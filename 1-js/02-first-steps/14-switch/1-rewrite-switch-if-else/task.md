# Omskriv "switch" til "if"

Skriv koden ved hjælp af `if..else`, som svarer til følgende `switch`:

```js
switch (browser) {
  case 'Edge':
    alert( "Du bruger Edge!" );
    break;

  case 'Chrome':
  case 'Firefox':
  case 'Safari':
  case 'Opera':
    alert( 'Okay vi understøtter også disse browsere' );
    break;

  default:
    alert( 'Vi håber, at denne side ser godt ud!' );
}
```

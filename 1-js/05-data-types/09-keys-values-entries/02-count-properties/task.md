# Tæl egenskaber

Skriv en funktion `count(obj)`, der returnerer antallet af egenskaber i objektet:

```js
let user = {
  name: 'John',
  age: 30
};

alert( count(user) ); // 2
```

Prøv at gøre koden så kort som muligt.

P.S. Ignorer symbolske egenskaber, tæl kun "almindelige".

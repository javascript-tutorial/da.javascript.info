# Vis dagen i ugen

Skriv en funktion `getWeekDay(date)` der viser dagen i ugen i forkortet format: 'Man', 'Tir', 'Ons', 'Tors', 'Fre', 'Lør', 'Søn'.

For eksempel:

```js no-beautify
let date = new Date(2014, 0, 3);  // 3. januar 2014
alert( getWeekDay(date) );        // Skal skrive "Fre
```
